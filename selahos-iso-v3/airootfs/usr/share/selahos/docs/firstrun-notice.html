Known Limitations and Workarounds

⚠ Ableton Live 12 Lite (Windows)

Status: Crashes on launch with onnxruntime_abl.dll not found error.

Why: Ableton's AI runtime (ONNX) is not currently functional in Wine.

Workaround: Reinstall Ableton 12 Lite from your original installer. The DLL must be present even if disabled. Alternatively, use Reaper or FL Studio (both fully working) for V1 Beta. We're tracking a proper fix for V1.1.

⚠ Electron-based AI Desktop Apps

Affected: ChatGPT Desktop (Windows), Claude Desktop (Windows), and similar Electron apps using newer Node.js.

Status: Fail to launch with EBADF stderr error.

Workaround: Use the web versions in Firefox at chat.openai.com and claude.ai. Web versions provide full functionality.

⚠ Apple Internal WiFi (Broadcom BCM4350)

Status: Internal WiFi on 2017 MacBook Pro and similar Macs does not function with linux-zen kernel.

Workaround: Use a class-compliant USB WiFi adapter ($15-25). Examples: TP-Link Archer T2U Plus, Alfa AWUS036ACS, or any USB WiFi marketed as "Linux compatible." This is a kernel limitation, not a SelahOS bug.

⚠ Apple Internal Audio Codec (CS8409)

Status: Internal speakers and microphone on certain Mac models not functional.

Workaround: Use a USB audio interface or USB microphone. Headphone jack works on most affected models. Native driver work tracked for V1.1.

⚠ Apple FaceTime HD Camera

Status: Internal camera not detected on most Mac models out of the box.

Workaround: Use a USB webcam, or install the community facetimehd firmware (advanced — may not work on all kernels). Tracked for V1.1.

System Care

Memory and Performance

SelahOS includes automatic memory tuning optimized for creator workloads:

  • zram compressed swap (configured automatically based on your RAM)
  • Lower swappiness (10 vs Arch default 60) to keep your work in RAM
  • Aggressive cache for project file scanning

To check current memory state, run selahos-memory-info in Konsole.

Recovery

If your install ever fails to boot (especially after a Mac NVRAM reset):

  1. Boot from your SelahOS USB stick
  2. Open Konsole
  3. Run: sudo selah-rescue
  4. Reboot — your install is restored

Updates

Updates arrive automatically via selah-update. Check the system tray for notifications. For manual update: open Konsole and run sudo pacman -Syu.
